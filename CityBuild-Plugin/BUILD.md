# CityBuild Maven Build

## Quick Commands

### Build Plugin
```bash
mvn clean package
```

### Install to local repo
```bash
mvn install
```

### Run tests
```bash
mvn test
```

### Generate docs
```bash
mvn javadoc:javadoc
```

---

## Output
- **JAR:** `target/CityBuildPlugin-1.0.0.jar`
- **Shaded JAR:** All dependencies included

---

## Requirements
- Java 21+
- Maven 3.9+
- Paper API 1.21.1
